# 3D Brick Breaker

OpenGL로 만든 3D 벽돌 깨기 게임입니다.  
3×3×3 구조의 블록 덩어리 중앙에 있는 **파란색 코어**를 공으로 파괴하는 것이 목표입니다.

## 플레이 화면

공이 팔각형 경계 벽 안에서 튀어다니며 블록을 깨고, 마우스 드래그로 블록 덩어리를 회전시켜 코어를 노출시켜야 합니다.

## 게임 모드

| 모드 | 설명 |
|------|------|
| **스토리 모드** | 3개 스테이지를 순서대로 진행. 최단 클리어 시간 기록 |
| **무한 모드** | 스테이지가 끝없이 생성. 최고 도달 웨이브 수 기록 |

## 조작 방법

| 입력 | 동작 |
|------|------|
| 마우스 좌클릭 + 드래그 | 3D 블록 덩어리 회전 |
| `A` / `D` | 팔각형 경계 벽 회전 |
| `Z` | 공 대시 필살기 (10초 쿨타임) |
| `1` | 스토리 모드 시작 |
| `2` | 무한 모드 시작 |
| `3` | 조작법 안내 화면 |
| `Enter` | 다음 스테이지 진행 / 메뉴 복귀 |
| `Esc` | 무한 모드 중 메뉴 복귀 |

## 실행 방법

`dist/` 폴더의 `Bruck_Breaker.exe`를 실행하면 됩니다. 별도의 DLL 설치가 필요 없습니다.

> `freeglut.dll`이 exe 내부에 내장되어 있어 자동으로 추출·로드됩니다.

## 빌드

**요구 사항:** Visual Studio 2022, Windows 10/11 x64

1. `Bruck_Breaker.slnx`를 Visual Studio에서 열기
2. 구성을 **Release | x64**로 설정
3. 빌드 (`Ctrl+Shift+B`)

| 빌드 설정 | 값 |
|-----------|-----|
| 런타임 라이브러리 | `/MT` (Static CRT) |
| 링커 옵션 | `/DELAYLOAD:freeglut.dll` |

## 기술 스택

- **언어:** C++20
- **그래픽:** OpenGL / FreeGLUT / GLU
- **IDE:** Visual Studio 2022
